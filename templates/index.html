Features


                    

                        
Workflow Tab

                        
    
                            
  • Browse, search, and filter your workflow collection
    • 
                            
  • Import workflows from JSON or PNG files
    • 
                            
  • Organize workflows into groups
    • 
                            
  • View thumbnails, JSON details, and properties
    • 
                            
  • Click a card to open the detail modal for analysis, tagging, and summarization
    • 
                        

                    


                    

                        
GenerateUI Tab

                        
    
                            
  • Load a workflow and edit parameters (prompts, images, models, settings)
    • 
                            
  • Connect to ComfyUI and generate images directly
    • 
                            
  • Control seed behavior (random, fixed, increment, decrement)
    • 
                            
  • View generated results with thumbnail history
    • 
                        

                    


                    

                        
Prompt Tab

                        
    
                            
  • Chat with AI assistant (Ollama) for prompt ideas and improvements
    • 
                            
  • Translate prompts between Japanese, English, and Chinese
    • 
                            
  • Save and manage prompt presets
    • 
                            
  • Apply presets directly to GenerateUI
    • 
                        

                    


                    

                        
Settings Tab

                        
    
                            
  • Configure ComfyUI connection URL
    • 
                            
  • Set up Ollama API for AI assistant features
    • 
                            
  • Change UI language and summary language
    • 
                            
  • Configure workflow data folder and Eagle integration
    • 
                        

                    


                    

                        
Tips

                        
    
                            
  • Drag & drop PNG files with embedded workflow data to import them
    • 
                            
  • Use the star button to mark favorite workflows for quick access
    • 
                            
  • Set a default workflow to auto-load on startup from the detail modal
    •
